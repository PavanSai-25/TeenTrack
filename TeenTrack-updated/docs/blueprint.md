# **App Name**: Teen Track

## Core Features:

- AI Money Motivator Tool: An AI tool that analyzes user progress to generate personalized, snappy motivational quotes and savings advice tailored for teens.
- Glass-Card Dashboard: A visually stunning home screen with semi-transparent frosted cards displaying total balance and monthly summaries with glowing text.
- Level-Up Progress Badge: A dynamic pill-badge that reflects user 'Money Master' status through pulsing animations as milestones are reached.
- Smart Ledger & Persistence: Form-based input for tracking allowance and spending, with immediate visual feedback and browser-level data persistence.
- Gamified Locked Savings: A high-stakes visual vault for time-locked funds that features interactive countdown timers and bonus percentage tracking.
- Financial Achievement Gallery: An interactive list of collectible badges that slide into view when the user completes specific spending or saving tasks.
- Category Insight Grid: A 4-column grid of animated cards for gaming, food, shopping, and miscellaneous expenses with diagonal hover sweep effects.

## Style Guidelines:

- A vibrant dark scheme featuring a Primary purple (#764ba2) that pairs with a Deep Navy desaturated background (#121017) and high-energy Teal (#4ecdc4) for accent highlights.
- 'Poppins' for all headlines and UI labels to provide a geometric, fashion-forward look; 'PT Sans' for smaller paragraph blocks for high legibility.
- Playful, high-detail emojis with 'floating' and 'bouncing' keyframe animations to inject energy into every financial card.
- Glassmorphism architecture using backdrop-blur (20px) and semi-transparent white containers (rgba(255, 255, 255, 0.15)) for a high-end feel.
- Smooth 8-second diagonal gradient cycles for the body background, combined with pill-shaped buttons that ripple and expand on interaction.
